开始上手(搭建博客的过程)

  1. 申请一个 github,做仓库repository
  2. #git branch gh-pages //在master分支下面新建一个gh-pages分支
  3. #git checkout gh-pages //切换到新建的gh-pages分支
  4. #git rm -rf . //由于master分支和gh-pages分支里面的内容完全不同，所以首先清空了所有文件
  5. #git add . //跟踪自己定义的文件
  6. #git commit -m "Create gh-pages branch via Github" //提交本次修改
  7. #git checkout -b gh-pages origin/gh-pages //切换分支
  8. # git push origin gh-pages

学习反思

人要经常反省。

比如这个 [博客]（http://www.cnblogs.com/yuanzm/p/3945814.html#commentform） 博主，把事情分析地清楚，我才懂得。也是懂得，才能清楚。不可以再一股脑地蛮干了，要耐心，静心，查资料，分析。不能做伸手党。

也不是你准备做什么东西，什么也准备好了，然后等你轻轻挥挥手。你就伟大了。并不是这样子的。一句话是，不要忘记自己的梦想。

对于程序这条路，我的感觉是，不要怕辛苦，经常看英文文档。（你并没有做了什么，你要知道）对于前辈的话语，良药苦口利于病。反省自己是为了什么，为了完成任务么？还是。。。
