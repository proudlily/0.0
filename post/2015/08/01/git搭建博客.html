用github搭建博客过程

榴白 ( 更新) git

开始上手

  1. 用Ink,生成静态的html.
  2. 申请github的帐号
  3. 在github上，申请博客的帐号。这个可以百度地到。
  4. 下面是比较重要的部分，网上没有讲到的。
    • $ git branch gh-pages,新建gh-pages分支
    • $ git checkout gh-pages,切换到 gh-pages分支
    • $ git rm -rf .,清空gh-pages内的内容
    • $ git add .,跟踪自己定义的文件
    • $ git commit -m "...",提交本次的修改
    • $ git checkout -b gh-pages origin/gh-pages 切换分支（可以不做）
    • $ git push origin gh-pages 远程更新

反思

人要经常反省。

比如这个博客，把事情分析地清楚，我才懂得。也是懂得，才能清楚。不可以再一股脑地蛮干了，要耐心，静心，查资料，分析。不能做伸手党。

也不是你准备做什么东西，什么也准备好了，然后等你轻轻挥挥手。你就伟大了。并不是这样子的。一句话是，不要忘记自己的梦想。

对于程序这条路，我的感觉是，不要怕辛苦，经常看英文文档。（你并没有做了什么，你要知道）对于前辈的话语，良药苦口利于病。反省自己是为了什么，为了完成任务么？还是。。。
